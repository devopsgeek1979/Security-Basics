## Option A: 14-Day Fast Track

### Week 1 (Core Concepts)

- **Day 1:** `topics/01-security-foundations.md` + glossary review
- **Day 2:** `topics/02-network-security-basics.md`
- **Day 3:** `topics/03-web-security-owasp.md` (OWASP Top 10)
- **Day 4:** `topics/04-identity-and-access-management.md`
- **Day 5:** `topics/05-cryptography-basics.md`
- **Day 6:** Revision + practice `interview/01-fundamentals-qa.md`
- **Day 7:** Mock interview (45 min)

### Week 2 (Practical + Interview Depth)

- **Day 8:** `topics/06-vulnerability-management.md`
- **Day 9:** `topics/07-cloud-security-basics.md`
- **Day 10:** `topics/08-devsecops-basics.md`
- **Day 11:** Practice `interview/02-network-and-web-qa.md`
- **Day 12:** Practice `interview/03-cloud-and-devsecops-qa.md`
- **Day 13:** Practice `interview/04-hr-and-behavioral-security-qa.md`
- **Day 14:** Full mock panel interview (60 min)

## Option B: 4-Week Deep Prep

- **Week 1:** Foundations + IAM + Crypto
- **Week 2:** Network + Web + OWASP
- **Week 3:** Vulnerability Management + Cloud Security
- **Week 4:** DevSecOps + Scenario Q&A + Mock interviews

## Daily Routine (Recommended)

- **45 min:** Concept study
- **30 min:** Video learning from `resources/youtube-references.md`
- **30 min:** Write answers in your own words
- **15 min:** Recap and flashcards

## Interview-Ready Checklist

- [ ] Can explain CIA triad with real example
- [ ] Can explain authentication vs authorization
- [ ] Can explain HTTPS handshake at high level
- [ ] Can explain OWASP Top 10 and mitigations
- [ ] Can explain least privilege and MFA implementation
- [ ] Can discuss one real breach and lessons learned
- [ ] Can answer 20+ scenario-based questions confidently
