# Security Interview Cheat Sheet

## One-Line Definitions

- **CIA triad:** Confidentiality, Integrity, Availability.
- **Risk:** Likelihood times impact.
- **Least privilege:** Minimum permissions for required tasks.
- **Zero Trust:** Verify explicitly, assume breach.
- **Defense in depth:** Multiple independent controls.

## Top Practical Controls

- Enable MFA for all privileged users.
- Patch critical vulnerabilities quickly.
- Remove public exposure for sensitive services.
- Use centralized logging and alerting.
- Rotate keys/secrets regularly.

## 30-Second Interview Framework

1. Define concept simply.
2. Give real-world example.
3. Mention one tool/control.
4. Mention one trade-off/limitation.

## Incident Response Quick Steps

1. Detect
2. Contain
3. Eradicate
4. Recover
5. Lessons learned
